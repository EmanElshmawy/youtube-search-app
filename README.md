## How to create youtube APi Key 

1- Open google developer console
2- Create new project
3- Enable Api services
4- Search on youtube Apiv3
5- Click Enable 
6- Click on create credentials option + create credentials
7- Click on apikey then copy it 
8- It is ready to use , put in the environment.ts file
